# citizen-web

[2020 스마트시민랩 커뮤니티 지원사업] 센서를 통한 공공공지의 흡연량 연구 ‘동행'

[Citizen 웹사이트 가기](https://citizen-dongheng.tk/)

### 소개
시민 모두를 위해 시민들이 직접 참여하는 센서 저널리즘의 첫 시작을 함께했습니다.
- 아두이노 센서를 통해 공공 공지의 흡연량을 측정
- 차트와 지도를 통해 데이터 시각화
